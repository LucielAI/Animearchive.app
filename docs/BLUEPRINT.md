# Anime Architecture Archive — Project Blueprint

The Anime Architecture Archive is a structural intelligence platform designed to analyze fictional universes as systems.

## Core Idea
The archive is not intended to be encyclopedic.
It is intended to reveal the structural mechanics of fictional worlds.

Each universe is treated as a system defined by:
- rules
- power mechanics
- causal events
- faction dynamics
- counterplay structures

## Dual Voice Architecture
Every entity has two parallel descriptions:

LORE Mode
Human narrative explanation.

SYS Mode
System-level explanation framing the universe like an engineered system.

Example:

LORE:
Eren becomes the catalyst for the Rumbling.

SYS:
A deterministic execution node triggers a global catastrophic cascade.

## Core Product Flow

User experience funnel:

1. Immediate understanding → SystemSummary
2. Visual hook → renderer wow moment
3. Exploration → multi-tab dashboard
4. Discovery → explore another universe
