# Anime Database

Official Page: https://animearchive.app/

This is the anime database repository for the Anime Archive application.

## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Technologies

- React
- Vite
- ESLint
